# MCP Tooling Plan

## Recommended MCPs for this project

### Install First
- GitHub
- Context7
- Playwright
- Chrome DevTools MCP
- Desktop Commander
- Markitdown

### Install When Backend/Billing Starts
- Supabase
- Stripe

### Install Later If Needed
- Firecrawl
- Tavily
- Notion

## Why
### GitHub
Use for:
- repo access
- issues
- PR workflow
- file changes

### Context7
Use for:
- current framework and library docs
- Expo docs
- React Native docs
- FastAPI docs
- Stripe docs

### Playwright
Use for:
- end-to-end testing
- flow automation
- regression checks

### Chrome DevTools MCP
Use for:
- frontend debugging
- network inspection
- layout issues
- perf inspection

### Desktop Commander
Use for:
- file ops
- terminal commands
- process management

### Markitdown
Use for:
- converting source docs/assets into markdown

### Supabase
Use if you choose Supabase for:
- auth
- db
- storage
- realtime

### Stripe
Use for:
- subscriptions
- credits
- billing flows
- checkout setup
