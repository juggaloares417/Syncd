## Mic Match
Mic Match is a rotating music-based speed connection feature.

### Core Flow
1. Enter Games
2. Open Mic Match
3. Choose mode
4. Set preferences
5. Join queue
6. Enter lobby
7. Countdown
8. Pair into round
9. Prompt/challenge
10. Quick chat
11. Feedback
12. Rotate
13. Session summary

## Mic Match Modes
- Dating
- Friends
