# Syncd Web Stub

This folder contains a lightweight Vite-powered browser stub for Syncd.

## What it is
- A fast web sidecar for the existing mobile scaffold
- A place to shape discovery, matches, chat, and Karaoke Roulette flows in a browser
- A clean starting point for future shared UI and shared mock-data work

## Run it

```bash
cd web
npm install
npm run dev
```

## Current scope
- Dashboard shell
- Discovery and Karaoke Roulette overview blocks
- Mock matches
- Build target checklist
